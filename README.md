# Vehicle Detection using YOLOv11
An optimized computer vision pipeline for real-time vehicle detection from image URLs.

## Features
- High-accuracy detection of cars, trucks, buses, and motorcycles.
- Optimized for Google Colab with GPU acceleration.
- Clean visualization with minimal text overlays.

## Technology Stack
- **Language:** Python
- **Model:** YOLOv11 (Ultralytics)
- **Libraries:** OpenCV, NumPy, Requests
